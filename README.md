# Linker Frontend

Linker is a social networking platform that combines features. Users can discover profiles, show interest, ignore profiles, manage connection requests, and build meaningful professional/social connections through an interactive and responsive interface.

## 🚀 Live Demo

[Live Website](http://3.110.30.144/)

---

## ✨ Features

### 🔐 Authentication
- Login
- Signup
- Logout

### 👤 Profile Management
- Edit profile information
- Upload/view profile photo
- Change password
- Delete account

### 🤝 Connection System
- Interested / Ignore profiles
- Accept / Reject requests
- View sent requests
- View received requests
- View connections

### 🎨 UI Features
- Fully responsive design for mobile and desktop
- Light & Dark theme support
- Clean and modern interface

---

## 🛠️ Tech Stack

- React
- Vite
- Tailwind CSS
- DaisyUI
- Axios
- dnd-kit

## 📦 Installation

Clone the repository:

```bash
git clone <your-frontend-repository-url>
```

Move into the project directory:

```bash
cd linker-frontend
```

Install dependencies:

```bash
npm install
```

---

## ▶️ Run Locally

Start the development server:

```bash
npm run dev
```
